# Mecanocat
Aplicació per millorar la mecanografia en català

## Com fer servir

1. Obrir el fitxer anomenat "MECANOCAT.exe" i esperar uns 15-20 segons. Posar el nom del teu perfil a la consola quan t'ho demani,
t'hauràs de recordar de quin nom has posat(accents, majúscules, etc.) perquè és amb aquest nom amb el qual podràs accedir
a les teves dades/estadístiques. "MECANOCAT.exe" ha d'estar al mateix directori que "Stats.py", "README.txt", "MyDict.txt",
"MECANOCAT.py" i "GettingDictionary.py". 

2. Seleccionar la dificultat a l'esquerra, està seleccionat per defecte a la més baixa, 25%. La dificultat limita la varietat de
paraules que poden aparèixer, de les 500 a les 2000 paraules més comunes del català. La dificultat és personal, així que millor que 
començeu al 25% i anar pujant a mida que us sentiu còmodes amb cada nivell.

3. Escriure les paraules que surten al centre de la pantalla. Les paraules són seleccionades aleatoriament entre el grup 
seleccionat segons la dificultat. Per passar de línia has de prémer espai.

4. L'objectiu és millorar la precisió no la velocitat, així que si t'equivoques, pots tirar enrere i corregir-lo o continuar,
l'error comptarà igual. Si vols reiniciar la línia pots prémer "esc".

5. Quan hagis fet unes quantes línies pots prémer el botó de la dreta "gràfica del progrés" i veurás els caràcters per 
minut contraposats amb els errors. Això és perqué individualment cadascú vegi el seu progrés al llarg del temps.
